Privacy Policy

Last updated · 21 June 2026

The short version

  • No accounts. No ads. No analytics.
  • Your habits stay on your phone.
  • One tiny request for a daily quote — nothing about you is sent.

What we store

Habits, streaks, badges, notes, and settings live in a local database on your device. We never see them. Uninstalling the app removes them.

Backups

"Save backup" creates a JSON file you choose where to put. We don't see or store it.

The one network call

Each time you open the app we fetch one motivational quote from the public ZenQuotes service. The request is anonymous — no identifiers go with it.

Permissions

  • Notifications — so reminders pop up.
  • Exact alarms — so reminders fire on time.
  • Internet — only for the daily quote above.
  • Boot — so reminders survive restarts.

You can revoke any of these in Android Settings.
